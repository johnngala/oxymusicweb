# oxymusicweb

1.Download the Zip File or clone the project into an empty folder.
2.Do a 'npm i'  on the root directory to install the server side dependencies.
3.cd into the client folder and also do an 'npm i' to install the client side dependencies as well.
4.Finally do an 'npm run dev' to run both the client and server with concurrently.

Thats it!!!
